Develop a web page using React to show the state of any given stock. The page will have:

Three text fields:
One for the stock’s symbol
Two for minutes and seconds
A button to submit a search.
An area to display the stock data after the user submits a search.

The page will return the various data of the symbol’s stock and will refresh whenever the timer finishes. The data should include:
Open price of the day
High price of the day
Low price of the day
Current price
Previous close price
Time of the entry
